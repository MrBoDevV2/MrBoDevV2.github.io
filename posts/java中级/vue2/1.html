                                                    # 第1章 环境搭建

                                                    # 一.课程简介

                                                    本套课程为 Vue.js 简洁实用版本课程，没有长篇大论。不需要大量时间学习。我们致力于用
                                                    最少的时间搞定最实用的部分。针对于需要快速搞定 Vue.js 的同学提供一个最方便的学习方
                                                    式

                                                    # 二、适合人群

                                                    1.前端初学者快速入门
                                                    2.做后台的同学有前端框架需求
                                                    3.初学者着重于实现效果的同学

                                                    # 三、课程重点包含内容

                                                    1.Vue 基础内容
                                                    2.Vue 组件详解
                                                    3.Vue 路由配置
                                                    4.网络请求、跨域与封装
                                                    5.Vue 状态管理
                                                    6.Vue 核心插件应用
                                                    7.Vue 打包部署

                                                    # 四.安装相关依赖

                                                    # 1.安装node https://nodejs.cn/download/open in new window

                                                    # 2. 验证是否安装成功

                                                    # 3.安装npm镜像cnpm:

                                                    # 4.下载速度较慢切换为cnpm

                                                    npm config set registry https://registry.npm.taobao.org

                                                    # 5.查看是否切换成功

                                                    npm config get registry

                                                    # 6.也可以直接安装cnpm

                                                    npm install -g cnpm -registry=https://registry.npm.taobao.org
